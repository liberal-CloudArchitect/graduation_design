## 一、项目概述

### 1.1 项目目标

构建**"从阅读到写作"全流程智能辅助**的学术文献分析平台：

| 功能模块 | 描述 | 核心技术 |
|----------|------|----------|
| 智能解析 | 精准提取PDF文献结构化信息 | LayoutLMv3 + ChatGLM |
| RAG问答 | 基于文献的自然语言问答 | LangChain + Milvus |
| 趋势分析 | 研究热点识别、突现词检测 | CiteSpace + ECharts |
| 写作辅助 | 论文大纲生成、文献综述 | GPT-4o / DeepSeek-V3 |
| **Agent记忆** | 智能记忆管理与多Agent协作 | Milvus + LLM重构 |

#### 核心代码：动态记忆集成

```python
# qa_engine.py 修改示例
class RAGQAEngine:
    def __init__(self):
        self.memory_engine = DynamicMemoryEngine(milvus, embedder)
    
    async def answer(self, question: str, project_id: int) -> dict:
        # 1. 检索历史记忆
        memory_results = await self.memory_engine.retrieve(question, project_id, top_k=3)
        
        # 2. 检索文献片段
        docs = await self.vector_store.search(...)
        
        # 3. 融合记忆与文献
        context = self._build_context_with_memory(docs, memory_results)
        
        # 4. 生成答案
        answer = await self.llm.ainvoke(...)
        
        # 5. 记录本次交互
        await self.memory_engine.add_memory(
            content=f"Q: {question}\nA: {answer}",
            metadata={"project_id": project_id, "agent_source": "qa_agent"}
        )
        
        return {"answer": answer, "memory_used": len(memory_results) > 0}
```
